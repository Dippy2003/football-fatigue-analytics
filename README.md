# PlayerPulse

PlayerPulse is an explainable football workload, fatigue-indicator, and
performance-risk analytics platform for coaches, analysts, students, and
portfolio reviewers.

## Ethical scope

PlayerPulse provides performance-based decision-support indicators from the
match data available to it. It does not diagnose fatigue, predict a confirmed
injury, replace qualified medical or sports-science assessment, or recommend
treatment.

> PlayerPulse provides performance-based indicators from available match data.
> It is not a medical diagnostic tool and must not be used as a substitute for
> qualified medical or sports-science assessment.

## Development status

The repository is being built in five tagged phases. Day 1 establishes the
application shell, health API, reproducible environments, database foundation,
tests, containers, CI, and documentation. See `docs/PROGRESS.md` for the exact
checkpoint.

## Data and licence boundary

The root MIT licence covers original PlayerPulse code and documentation only.
It does not grant rights to any third-party dataset, provider name, trademark,
logo, club badge, player image, or other external material.

The public demo, tests, CI, and screenshots use deterministic fictional data
generated by PlayerPulse. Metrica Sports sample data is local-import-only under
the project's conservative policy. StatsBomb Open Data is optional and remains
subject to its current official licence and attribution requirements. No raw
third-party football data is distributed by this repository.
