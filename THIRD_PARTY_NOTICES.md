only. It does not cover external datasets, trademarks, logos, photographs,

## Football data sources

### Metrica Sports sample data

- Official source: https://github.com/metrica-sports/sample-data
- Official usage statement: https://github.com/metrica-sports/sample-data#legal-stuff
- Project policy: optional local import only; never mirrored, committed,
  included in images, or deployed without separately verified permission.
- Required public acknowledgement: Metrica Sports.

### StatsBomb Open Data

- Official source: https://github.com/hudl/open-data
- Official licence: https://github.com/hudl/open-data/blob/master/LICENSE.pdf
- Project policy: optional rights-gated adapter; never mirror the complete
  dataset by default and re-check current conditions before any use or release.
- Required attribution: StatsBomb, including the current publication branding
  requirement described by the official repository.

No provider logo is bundled at this checkpoint. Text attribution and links are
used until a current provider requirement and permitted media-pack use are
verified for the relevant publication.
